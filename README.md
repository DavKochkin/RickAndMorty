# Rick And Morty App 

### Description 
Full featured universal ( iOS, MacOS, iPadOS) app The Rick And Morty API.

The app has 5 key area: 
- Characters
- Locations
- Episodes
- Settings
- Search

# Overview 

+ Written in SWIFT
+ Designed with MVVM pattern
+ Uses Both UIKit and SwiftUI
+ No External Dependecies
+ Pagination Support for data
+ Auto layout Based
